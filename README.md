# Intelligence Amplification

A static, text-only web edition of the Intelligence Amplification living booklet.

The site is intentionally simple: a single `index.html` file, anchored sections, no artwork, no build step, and no framework dependency.

## Local preview

Open `index.html` directly in a browser.

## Deployment

This repository deploys to GitHub Pages at https://intelligenceamplification.github.io/.
